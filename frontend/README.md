# JobColony - Flutter MVP (Riverpod)


This Flutter project is an MVP mobile client for the JobColony FastAPI backend.
It uses Riverpod for state management and `http` for network calls.


**Backend base URL**: set `const backendBase = 'http://10.0.2.2:8000'` (Android emulator) or your server URL.


Run:
- flutter pub get
- flutter run
